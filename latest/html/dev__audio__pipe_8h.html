enum rt_audio_pipe_flag

Aduio pipe flags.

Enumerator
RT_PIPE_FLAG_NONBLOCK_RDWR 

both read and write won't block

RT_PIPE_FLAG_BLOCK_RD 

read would block

RT_PIPE_FLAG_BLOCK_WR 

write would block

RT_PIPE_FLAG_FORCE_WR 

write to this pipe will discard some data when the pipe is full. When this flag is set, RT_PIPE_FLAG_BLOCK_WR will be ignored since write operation will always be success.
