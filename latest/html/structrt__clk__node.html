Detailed Description

Clk node, it is a pat of clk source or controller.

Note
Defined as the array like this if the CLK have multi out clocks:
struct XYZ_single_clk
{
struct rt_clk_node parent;
...
};
struct XYZ_multi_clk
{
struct rt_clk_node parent[N];
...
};
